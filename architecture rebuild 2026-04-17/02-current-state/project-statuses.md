## Work DB — Agent Rules (summary)

- **Hierarchy:** item with sub-items = Project (Health field); item without = Task (Status field)
- **Auto-create** (no permission needed): "I need to…", "I should…", "remember to…", committed deliverables
- **Auto-update** (no permission needed): status changes from conversation signals
- **Ask first:** archiving, deleting, reparenting, schema changes
- **Routing:** Creator Engine = content Edmund produces; Swipe Files = external inspiration; Work DB = tasks/projects; SOPs = workflows
